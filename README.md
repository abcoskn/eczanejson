# eczanejson
## Açıklama
İstanbuldaki güncel nöbetçi eczane bilgilerini JSON formatında çıktı veren bir php api.
Elde edilebilen eczane bilgileri;
* Eczane İsmi
* Eczane Telefon Numarası
* Eczane Adresi
* Eczane Yol Tarifi
* Eczane Koordinatları
